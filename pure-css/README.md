# 纯css的图片
## 关于 scss 文件
### 安装 [sass](https://www.runoob.com/sass/sass-install.html)
#### NPM 安装
- 我们可以使用 npm（NPM 使用介绍）来安装 Sass。
```npm
npm install -g sass
```
#### 将 .scss 文件转化的 css 代码
- cmd 使用管理员权限，进入 `**/*.scss` 文件目录
```
sass input-file-name.scss output-file-name.css
```
